# ChangeLog

## [1.0.0] - 2026-06-23

### Added

- MVP版本发布，扩展Jena Fuseki服务，提供可视化图谱
- SPARQL查询编辑器：语法高亮、关键词自动补全、格式化、复制单行
- SPARQL编辑器支持简单语法检查（实时lint提示）
- SPARQL查询结果支持分页展示，超大结果集警告
- SPARQL查询结果支持导出为CSV文件
- SPARQL查询结果支持多选行，导出为JSON（格式化/压缩/SPARQL原始格式），支持复制和保存文件
- SPARQL查询历史记录（本地持久化，支持快速回填和一键清空）
- SPARQL编辑器左右分栏支持拖拽调整宽度
- 图谱可视化页面：节点展开、邻居查询、关键词搜索、SPARQL转图谱
- 数据集管理页面：查看Fuseki服务状态、数据集列表
